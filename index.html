TutorDock 2.0

Internal Management Tool.

Admin

Tutor

Tutordock (TD) is a web application written in Python back-end and JavaScript for the front-end using AngularJS. One of the main uses of TD is to keep tracking of all the companies’s financials. For example, some of the examples one is able to see is: how many new clients signed up, how much revenue was generated, and to see how many concurrent programs are in effect. Company staff is able to create new tutoring programs for students and all billing and lesson notes are automatically sent to the family. One of the most exciting features of Tutordock is the ability to search and let multiple tutors know that a program is available. This makes it very fast to match students with potential tutors compared to other test prep companies that have to manually reach out to each tutor. One of the newer features is a built test scoring system, which generates a report breakdown of a student’s practice tests based on their answers. One of the most used features is the lesson tracking system where tutors and administrators keep track of all student lessons. Nonetheless, the main purpose of Tutordock is to assist tutors in managing their students learning.
